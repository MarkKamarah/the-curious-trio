# Quick GitHub Setup

## Run These Two Scripts:

1. **setup-git.bat** - Sets up git repository
2. **push-to-github.bat** - Pushes to GitHub

## Authentication

When pushing, you'll need a Personal Access Token:
1. Go to https://github.com/settings/tokens
2. Generate new token (classic)
3. Select "repo" scope
4. Copy the token
5. Use it as your password when prompted
